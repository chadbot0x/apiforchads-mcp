# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-23

### Added
- Initial release with 9 MCP tools
- Real-time crypto prices via Chainlink oracles + Binance
- Polymarket CLOB orderbook data with slug-to-token resolution
- Quick research (Gemini Flash + Google grounding, ~20s)
- Deep research (autonomous agent, ~5min)
- JS-rendered web scraping (markdown, text, HTML)
- Full-page screenshots (PNG)
- CSS selector extraction
- Page-to-PDF conversion
- x402 Solana micropayment support
- API key authentication
- Streamable HTTP MCP transport
- Rate limiting per endpoint
- SSRF protection on render endpoints
- Claude Desktop, Cursor, and Windsurf configuration examples
